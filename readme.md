# Udemy Course Data Analysis (EDA Project)

## Project Overview
- This project performs Exploratory Data Analysis (EDA) on a Udemy courses dataset to uncover insights related to course trends, pricing strategies, and subscriber behavior.
- The analysis focuses on understanding how different factors such as course category, price, level, and content duration influence course popularity.
- The objective is to derive actionable insights that can help instructors and platforms optimize their offerings.

---

## Problem Statement
- With thousands of courses available online, identifying what makes a course successful is critical.
- This project analyzes Udemy data to determine key factors driving course performance.

### Key Questions:
- Which categories have the highest number of courses?
- What type of courses attract the most subscribers?
- How does pricing affect course popularity?
- Which course levels (Beginner, Intermediate, Expert) perform best?
- Is there a relationship between content duration and engagement?

---

## Dataset Description
- The dataset contains detailed information about Udemy courses, including:
  - Course Title
  - Subject/Category
  - Price (Free/Paid)
  - Number of Subscribers
  - Number of Reviews
  - Course Level
  - Content Duration
  - Published Date

### Data Characteristics:
- Combination of numerical and categorical variables
- Real-world dataset with potential inconsistencies and missing values

## Exploratory Data Analysis (EDA)

### Univariate Analysis
- Distribution of:
  - Course categories
  - Price (Free vs Paid)
  - Course levels
  - Content duration

### Bivariate Analysis
- Relationship between:
  - Price and number of subscribers
  - Course level and engagement
  - Category and popularity

### Multivariate Analysis
- Combined effect of:
  - Category, level, and price on subscribers
  - Duration and reviews on course success

---

## Key Insights
- Certain categories dominate the platform in terms of course volume
- Free courses tend to attract a higher number of subscribers
- Beginner-level courses have broader audience reach
- Course duration shows a moderate impact on engagement
- Pricing strategy significantly influences course performance
